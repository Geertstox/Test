How to use this tool

A practical workflow for discovering and qualifying online communities for engagement.

1

Enter your keyword and location

Go to Search queries. Type the topic you are looking for (e.g. climate action) and the city or region (e.g. Brussels). Select the platforms you want to search.

2

Open each query in Google

Click the Open button next to each generated query. Google will return indexed public group pages, WhatsApp invite links, and Instagram accounts matching your search.

3

Evaluate each result manually

Visit each group page directly. Check member count, posting frequency, and whether the topic is genuinely relevant. Look for the Very active label on Facebook groups.

4

Log communities in the registry

Add each relevant community with its platform, location, topic, member count, and a relevance score (1–5). Set the status to Discovered initially.

5

Qualify and update status

As you research further, update each entry: Qualified once you have confirmed it is relevant and active, Contacted once you have reached out, Engaged once there is a relationship.

Platform notes
Facebook groups

The query site:facebook.com/groups "[city]" "[keyword]" returns indexed public groups. Add "very active" to filter for higher-activity groups. Member count is visible on the group page without logging in.

WhatsApp communities

WhatsApp groups cannot be searched directly. This tool finds publicly shared invite links (chat.whatsapp.com) that group admins have posted on Facebook, websites, or other platforms. Always verify the group is still active before using the link.

Instagram

Instagram has no group structure. The query finds public accounts matching your keyword and location. Supplement with hashtag and location tag searches directly inside Instagram.
